About Mud Pies

Mud Pies is a simple game inspired by Google Doodle Action Gnomes and Angry Birds, developed by the GitHub user live-by-unix as a fun project to pass time.

How to Play:

Click and drag the mud pie to aim and build up power, then release to throw it as far as possible! The game features dynamic wind conditions that affect your throw, and a real-time stopwatch to track how long your mud pie stays in the air.

Features:

  • Physics-based throwing mechanics
  • Dynamic wind effects
  • Score tracking with localStorage
  • Progressive Web App (PWA) support
  • Touch and keyboard controls
  • Offline playability
